WeatherList WeatherList provides realtime information on weather in the areas you care about. Simply search for a location based on its latituted and longitued, and you'll be served the latest weather data available. Weather.gov All weather data is sourced from the National Weather Service. More information on the National Weather Service can be found at forecast.weather.gov Technologies This application was developed with Angular 9, Java Spring Boot, and MongoDB.
